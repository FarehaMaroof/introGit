GIT AND GIT HUB

GitHub allows developers to create, store, manage, and share their code. Git is version control system and GitHub is a cloud-based platform where you can store, share, and work together with others to write code.

Steps to download and install Git

  1. Download Git from Download git
  2. Select the Type of Operating System and Download
  3. Restart or Reboot the PC
  4. To check if its installed go to VS code and open terminal from top left options.Type 'git --version' on terminal and it will show the latest terminal installed

To use git in VS

  1. In Terminal type command 'git init'.It will initialize in specific folder and keep track of everything in the project
  2. Now 'U' will appear after all the files in that specific folder representing untracked file i.e Git is not able to track their location.yoc can check the same by typing 'git status' it will give list of files that are untracked
  3. To track file write command 'git add filename' and to track all files type 'git add.' and the commit by typing 'git commit -m 'your msg'
  4. If you are Committing for the first time then you will get following error: *** Please tell me who you are. Run git config --global user.email "you@example.com" git config --global user.name "Your Name" to set your account's default identity. Omit --global to set the identity only in this repository. So you have to run these commands: git config --global user.email "you@example.com" git config --global user.name "Your Name" and configure it and then again add and commit. commit is used to create a log of the changes made so far and keeps track of the changes made so far.
  5. To see number of times committed type commands 'git log'

Suppose you made some mistake in last commit and you want to delete it and go back to last second commit then

  1. Type 'git log' command and see the log select the code of the commit you want to go back to eg:commit athe31678ba550gh109eabfd0da4588f3c6 now copy the code 'athe31678ba550gh109eabfd0da4588f3c6'
  2. type command and paste the code : 'git reset --hard athe31678ba550gh109eabfd0da4588f3c6' hard :when you want to completely remove last version type command and paste the code :'git reset --soft athe31678ba550gh109eabfd0da4588f3c6' soft :When you want to go to required version bt doesnt want to remove the version in which mistake was made.
